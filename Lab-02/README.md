# Lab 02: View the Switch MAC Address Table  

## Objective: To build and Configure a Network and examine the Switch MAC Address Table 

## Built and Configured a Network 
* Created the network topology. 
* Cabled the network according to the topology. 
* Configured PC hosts. 
* Initialized and reloaded switches as necessary. 
* Configured basic settings include paswwording for each switch (Console and vty passwords : Cisco, Priv EXEC password : class)

## Examined the Switch MAC Address Table.
* Viewed and recorded network device MAC addresses.
* Displayed the switch MAC address table.
* Established ping connections between all devices in the network and confimed registration of all required MAC address on the addresing table of individual switches.
