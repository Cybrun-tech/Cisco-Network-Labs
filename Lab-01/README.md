# Lab 01: Basic Switch Configuration, View Network Device MAC Addresses

## Objective
To perform initial management setup on a Cisco 2960 Switch & end device PC, verify Layer 2 connectivity and view device MAC addresses.

## Configuration & Connectivity 
   **Console Access:** Established a serial link from PC-A (RS-232) to the Switch (Console).
   **Hostname:** Assigned a hostname to the switch (S1)
   **DNS lookup:** Disabled DNS lookup to stop typos from freezing the CLI
   **Addressing:** Assigned ip addresses and subnet masks to the devices using the addressing table
   **Connectivity:** Verified network connectivity between the switch and PC

## Network Device MAC Addresses
   **OUI:** Viewed and verified the MAC OUI(Organizationally Unique Identifier) of the devices 
   **Serial Number:** Viewed the MAC serial numbers of the devices
   **NIC Manufacturer:** Searched the vendor that manufactured the NIC
   **The BIA:** Identified the burned-In address (Hardware MAC) and its purpose
   **MAC Address Table:** Viewed the MAC address of the PC on the switch MAC address table
